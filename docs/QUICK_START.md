# Quick Start

Get up and running with Modular Sticky development in less than a minute.

```bash
# 1. Clone the repository
git clone https://github.com/sanchez314c/modular-sticky.git

# 2. Enter the directory
cd modular-sticky

# 3. Install dependencies
npm install

# 4. Start the application in development mode
npm run dev
```

## What just happened?
`npm run dev` launches the Electron application with the `--dev` and `--no-sandbox` flags. You should see the Neo-Noir Glass Manager window appear on your desktop.

## Next Steps
- Try adding a new note from the Manager window.
- Check `main.js` to see the entry point.
- Check `renderer.js` and `note.html` to modify the note UI.
