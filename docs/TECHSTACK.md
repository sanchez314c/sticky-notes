# Tech Stack

- **Application Framework**: Electron (v37.4.0)
- **JavaScript Engine**: Node.js (Main Process) / Chromium V8 (Renderer)
- **Frontend Core**: Vanilla JavaScript (ES6+), HTML5
- **Styling**: CSS3 with CSS Variables, Neo-Noir Glass Aesthetics
- **Unique Identifier Generation**: `uuid` (v11.1.0)
- **Packaging & Distribution**: `electron-builder` (v26.0.12)
- **Linting**: ESLint (v8.57.0)
- **Formatting**: Prettier (v3.0.0)
- **Type Checking**: TypeScript (via JSDoc `tsc --noEmit`)
