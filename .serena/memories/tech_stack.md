# StickyNotes - Technology Stack

## SWARM Framework Technology Support
- **Primary Platform**: Multi-platform via various tech stacks
- **Build Systems**: Electron, Swift (macOS/iOS), Python, Web/React, TypeScript
- **Languages**: JavaScript/TypeScript, Swift, Python, HTML/CSS
- **Platforms**: macOS, Windows, Linux, iOS, Web

## Available Build Configurations
- `electron-build-system.md` - Cross-platform desktop
- `swift-macos-build-system.md` - Native macOS
- `swift-ios-build-system.md` - iOS/iPadOS  
- `python-build-system.md` - Standalone executables
- `web-build-system.md` - Modern web apps
- `typescript-build-system.md` - Type-safe development

## Execution Environment
- **Shell**: Bash (macOS/Linux compatible)
- **Python**: 3.8+ required
- **Claude CLI**: Required for swarm execution
- **Screenshots**: Native system tools (`screencapture` on macOS)
