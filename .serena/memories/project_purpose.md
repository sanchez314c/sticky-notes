# StickyNotes Project - Purpose & Context

## Project Purpose
StickyNotes application to be built using the SWARM autonomous development framework.

## SWARM Framework Overview
- **Location**: `/Volumes/Development/Testing/StickyNotes/SWARM/`
- **Type**: Autonomous AI development system with parallel agent orchestration
- **Capability**: Complete app development from idea → PRD → code → build → test → deploy
- **Method**: Two-phase approach (80% rapid generation + 20% swarm repair)

## Key Components
- 6 specialized parallel agents (Executive, Architecture, UX/UI, Features, Performance, Competitive)
- Visual testing with screenshot analysis
- Real-time monitoring and accountability
- Multi-platform build systems (Electron, Swift, Python, Web)
- Automated error detection and self-healing
